VS CODE

O visual estudio code é uma ferramenta da Microsoft, que permite escrever cod em qualquer linguagem e tem por objetivo oferecer ferramenrtas que aumentem a velociadade de produçao de código.

Consideração

Antes de começar a utilizar o VSCODE, configure o ambiente seguindo alguns passos:

Ative a quebra de linha para q o cod sempre q atirgir o limite da lateral da linha faça a quebra. Isso pode ser feito usando o comando Alt+Z.

ative o autosalvamento:ative o autosalvamento para que os cod sejam salvos automaticamente.

Extensão liveServer:baixe a estensão para que os cod em HTML sejam exibidos e atualizados automaticamente

Atalho "!": esse atalho possibilita que o cod essencial do HTML no início do arquivo automatico
